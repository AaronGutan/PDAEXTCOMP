Общее описание

Внешняя компонента  LogAddIn.dll предназначена для работы «1С:Предприятие 8. Расширение для карманных компьютеров» и выполняет вывод необходимой информации в лог-файл. Компонента может быть использована, например, в целях отладки мобильных приложений.

 

Имя объекта для использования в мобильных приложениях: LogAddIn.

 

Свойства компоненты

 

Включен (IsEnabled)

 

Тип значения – число. Чтение и запись.

0 - выключен, 1 – включен

 

Содержит признак того, включен или выключен вывод информации в лог-файл.

 

ИмяФайла (FileName)

 

Тип значения – Строка. Чтение и запись.

Полное имя файла, в который выводится информация.

 

ПечататьВремя(PrintTime)

 

Тип значения – число. Чтение и запись.

0 – не печатать, 1 – печатать

 

Если установлено значение 1, то в файл выводится текущее время.

 

Методы компоненты

 

Вывести (Output)

 

Выводит в файл строку, переданную в качестве параметра.

 

Параметры:

ЗначениеВывода – значение, выводимое в файл. Обязательный.

Тип значения – Строка.

 

Удалить (Delete)

 

Удаляет лог-файл с диска.

 

УстановитьИмяФайла (SetFileName

 

Устанавливает имя лог-файла.

 

Параметры:

 

ИмяФайла – устанавливаемое имя файла. Обязательный.

Тип значения – Строка.

 

ОчищатьФайл – признак очистки файла. Не обязательный. Значение по умолчанию – 0.

Тип значения – Число.

0 – не очищать, 1 – очищать
