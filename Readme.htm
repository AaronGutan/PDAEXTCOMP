Описание поставки примеров реализации к технологии создания и использования внешних компонент для 1С:Предприятие 8. Расширение для карманных компьютеров.

В каталоге Source размещены исходные коды:

  • в каталоге ScanSymb размещен исходный код внешней компоненты ScanSymb.dll, реализующей работу со встроенным сканером штрихкодов на мобильных устройствах Symbol. Для успешной сборки и компиляции проекта требуется Symbol SDK. Описание внешней компоненты приведено в файле ScanSymb.htm Данная компонента может использоваться также в качестве примера разработки компоненты с собственным окном и обработкой сообщений Windows.
  • в каталоге LogAddIn размещен исходный код внешней компоненты LogAddIn.dll, реализующей запись лог файла. Описание свойств и методов компоненты приведено в файле LogComp.htm.
  • в каталоге AddIn размещен исходный код библиотеки, которая содержит набор базовых классов и процедур для работы с внешней компонентой. Фактически, она реализует всю технологию COM-объекта и интерфейсы внешней компоненты, что взаимодействие внешней компоненты с "1С:Предприятие 8.Расширение для карманных компьютеров". Данная библиотека может быть использована в качестве основы для создания внешних компонент. В частности, она использована для создания внешних компонент ScanSymb.dll и LogAddIn.dll. Описание порядка использования библиотеки при создании внешних компонент приведено в файле CompDev.htm.
